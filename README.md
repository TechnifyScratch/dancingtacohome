# Dancing Taco — Landing Page

Static HTML landing page intended for Vercel hosting (no backend).

## Deploy

- Import this repo into Vercel.
- Framework preset: **Other** (static).
- Build command: **none**
- Output directory: **/** (root)

## Customize

- Copy in a real OpenGraph image if you want PNG/JPG instead of SVG: `assets/og.svg`
- Update support email placeholders in `privacy.html` and `terms.html`.
- Replace the preview area in `index.html` with real screenshots when available.
